# Russell's International

Official website for Russell's International — a leading consultancy providing study abroad counseling, skill training, IELTS preparation, internship placement, and career services.

## Tech Stack

- **Framework:** React 18 + TypeScript (Vite)
- **Styling:** Tailwind CSS + shadcn/ui
- **Backend:** Supabase
- **Routing:** React Router v6

## Getting Started

```bash
npm install
cp .env.example .env   # fill in your Supabase credentials
npm run dev
```
